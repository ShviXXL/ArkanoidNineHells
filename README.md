# ArkanoidNineHells

ArkanoidNineHells is a college course project inspired by the game Arkanoid and the Nine Hells of Baator.

## Installation

### Linux

```
git clone https://github.com/ShviXXL/ArkanoidNineHells
cd ArkanoidNineHells

cmake .
make

./ArkanoidNineHells
```
